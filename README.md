# js-sorting-algorithms
JavaScript based Sorting Algorithm is used to rearrange a given array or list of elements in an order.


| Algorithms | Time Complexity    |
| :---:   | :---: | :---: |
| Selection Sort | O(n2)   |
| Bubble Sort | O(n)   |
| Insertion Sort | O(n)  |
| Quick Sort | O(n log(n))  |
